### Generate executable binaries
[Windows](https://docs.flutter.dev/desktop#windows)
```
flutter build windows
```
[MacOs](https://docs.flutter.dev/desktop#macos)

```
flutter build macos
```
[Linux](https://docs.flutter.dev/desktop#linux)
```
flutter build linux
```
